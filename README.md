# Portfolio

![Version](https://img.shields.io/badge/version-1.0.1-informational)
![Tests](https://github.com/elo-tidy/portfolio/actions/workflows/node.js.yml/badge.svg)

## Fonctionnalités

- Sections : introduction, compétences, parcours professionnel et formations
- Téléchargement de la version CV papier
- Mode sombre / clair
- Outils de customisation avec sélection de la couleur principale
- Section parcours professionnel et formations avec filtres sticky permettant de sélectionner uniquement les expériences professionnnelles, uniquement les formations ou d'afficher le tout
- Navigation sticky avec section courante active
- Liens directs sticky de contact (email, linkeding et du CV papier)
- Animations expérimentales CSS scroll-driven
- Responsive

## Stack

- Nuxt 3
- Tailwind 4
- Vercel
- Supabase
- Vitest
- Github Action

## Démo

[lombard-eloise-portfolio](https://lombard-eloise-portfolio.vercel.app/)

## Lancer le projet localement

```bash
git clone https://github.com/elo-tidy/portfolio.git
cd portfolio
npm install
npm run dev
```
